Nossa intenção com o projeto:

Criar um fórum personalizado, com identidade visual e estilo de arte baseado em um jogo eletrônico. O principal intuito do fórum é integrar usuários em temas cotidianos, hobbies, tendências e dentre outros tópicos. isso acontece através de abertura e interação em threads para os usuários, tendo a opção de editar e excluir os comentários. Nosso projeto tem o foco em 3 páginas principais: Página de login, para o acesso à página principal do fórum; caso seja o primeiro contato do usuário, temos uma página de cadastro para o preenchimento dos dados mais importantes. A principal página de acesso do usuário é a página do fórum, na qual toda a interação será feita, hospedando os diversos temas e conversas do fórum. As páginas são renderizadas e gerênciadas através do Flask, assim como a navegação entre elas. Com ajuda de um dicionário em python, que guarda o Email e a senha submetidos da página do cadastro, a validação dos usuários é feita pela página de Login que também acessa este dicionário. O Flask também se responsabiliza pela lógica de abertura de tópicos e edição e exclusão dos comentários.

Construção do projeto html:

  • Criação da página de login do fórum
  • Criação da página de cadastro do fórum
  • Criação da página principal, utilizada para responder outros usuarios e adicionar thread de assuntos
  • Construção do CSS para as três páginas
  • Construção do Javascript para a página de cadastro
  • Uso do framework Flask para a parte back-end do site
